---

### 1.2 Product Vision

A website that **feels as visually premium as the work it showcases.** Inspired by:

- **Vibhu Trivedi style** — Raw, expressive scrapbook aesthetic with texture, hand-drawn accents, grid paper backgrounds, masking tape elements, torn edges, and brutalist oversized typography
- **Javeria style** — Clean structural clarity, Z-pattern layout, soft blob shapes, muted warm backgrounds, outline fonts as anchors

The fusion produces a **uniquely cinematic creative platform** — one that is bold yet navigable, artistic yet structured.

**Brand Identity:**
- Primary Color: **Orange (#FF6B35)**
- Secondary Colors: **Black (#0A0A0A)** and **White (#F5F5F0)**
- Accent: **Yellow highlight effect** (à la Vibhu Trivedi)
- Design Feel: Modern · Tech-inspired · Minimalist · Creative · Cinematic

## 5. NON-FUNCTIONAL REQUIREMENTS

### 5.1 Performance
- Lighthouse Performance Score: **90+** on mobile and desktop
- Core Web Vitals: LCP < 2.5s | FID < 100ms | CLS < 0.1
- Image optimization: WebP format + lazy loading
- Video: Embed via YouTube / Vimeo (no self-hosted video)

### 5.2 SEO
- Server-Side Rendering (SSR) or Static Site Generation (SSG) for all pages
- Structured data / JSON-LD: Person, Product, Article schemas
- Sitemap.xml and robots.txt
- Open Graph + Twitter Card meta tags
- **Target keywords:** `video editor India` · `motion graphics freelancer` · `YouTube video editing service`

### 5.3 Mobile Responsiveness
- Mobile-first design approach
- Breakpoints: 320px | 768px | 1024px | 1440px
- Touch-friendly interactive elements (min 44px tap targets)

### 5.4 Accessibility
- WCAG 2.1 AA compliance
- Keyboard navigation support
- ARIA labels on all interactive elements
- Color contrast ratio: minimum 4.5:1

### 5.5 Security
- HTTPS enforced via SSL
- CSRF protection on contact form
- Input validation and sanitization
- Environment variables for all API keys (Razorpay, email service)
